# AgencyWeb - Projeto Landing Page - Curso SENAI

Projeto desenvolvido durante o curso de Desenvolvimento Front-End do SENAI.

O objetivo foi praticar conceitos de HTML5, CSS3 e organização de layout a partir de um protótipo no Figma.

---

## 🎯 Objetivo

Desenvolver uma landing page utilizando HTML e CSS, seguindo um protótipo criado no Figma.

---
